# Demo video script — 3.5 minutes

**0:00–0:25 · The hook (screen: rejection note from a real run).**
"This is the rejection I'd get from [organisation] — written by my own product. It read the funding brief, then read my CV line, and told me exactly what the funder couldn't see. Thirteen years of choreography, invisible on paper. That's the problem."

**0:25–0:55 · Problem + who it's for.**
"Creatives changing careers don't struggle with formatting — they struggle with translation. Resume AI polishes wording; polish is a commodity. The Creative Transition Engine extracts what experience actually demonstrates, profiles the specific receiver, and writes the bridge. Two users, one pipeline: a dancer moving into product, an artist applying for funding."

**0:55–2:20 · Live run (screen recording, real data).**
- Paste the messy voice-note story: "Real transcript, frustrations left in — that's where the material is."
- Scout: "Don't know which story matters? It reads the target first and interviews you."
- Experience Graph: "Every capability pinned to a verbatim quote of mine — the anti-fabrication gate."
- Receiver Profile: "Including what they're afraid of. A funder fears money with no legacy."
- Bridges + human gate: "I approve only what I can defend in an interview. I rejected [n] here — the system proposes, I decide."
- Translation + Critic report: "The Critic enforces a banned-pattern list built from things I actually hate — it caught [example] and rewrote it. Tone is a measured property here, not a vibe."
- Interview Armour: "Every claim ships with the evidence behind it. No resume tool can offer this, because none of them have traceability."

**2:20–2:50 · How it's built.**
"Six agents, orchestrated deterministically in code — no orchestrator agent, by design: bounded LLM tasks, hard output budgets, JSON repair, automatic retry. Target Analyst integrates web search. Deployed as a live URL. Production plan routes extraction to a cheaper model tier — a config table, not a meta-agent."

**2:50–3:20 · Evidence of product thinking.**
"Success metrics: zero Critic violations, zero fabrications, bridge acceptance over 60%, and a recognition check — would the practitioner recognise themselves? Run logs in the repo. This is v2: v1 was a single translation skill; testing it exposed the missing half — the receiver."

**3:20–3:30 · Close.**
"Every claim yours. Approved by you. In your voice. The Creative Transition Engine — link in the summary."
